## Project Overview

The **Hospital Management System (HMS)** is a comprehensive desktop application developed to streamline the administrative and clinical operations of healthcare facilities. Built using **Java** and the **Swing** framework, this system provides a robust interface for managing patient records, doctor schedules, and hospital resources efficiently.

The application replaces manual paper-based systems with a digital solution, ensuring data accuracy, faster information retrieval, and improved operational workflow for receptionists, administrators, and medical staff.

## Key Features

* **Patient Administration**: Full-lifecycle management of patient data, including registration, discharge summaries, and medical history tracking.
* **Doctor & Staff Management**: Maintenance of detailed profiles for doctors and nursing staff, including specialization, shift scheduling, and contact information.
* **Appointment Scheduling**: An organized booking system to manage patient appointments, reduce wait times, and optimize doctor availability.
* **Room & Ward Allocation**: Real-time tracking of room occupancy and availability to facilitate smooth patient admissions.
* **Intuitive Desktop Interface**: A native graphical user interface (GUI) built with Swing components for a responsive and familiar user experience.
